Position Types

There are 4 types of positioning we can do with CSS
  • Static
    • Browser default
  • Relative
    • Changes it's position based on it's default
  • Absolute
    • Changes it's position based on it's parent element
  • Fixed
    • Takes it's position from the browser window

Static Positioning

  • It's position it given by the browser
  • Cannot be changed
  • Not very useful, but good to know about when using absolute positioning.

Relative positioning

  • The browser moves the element based on it original position
  • use position:relative; in css for element

Note about Absolute position

  • Absolute positioning look for the parent whose position is not static
  • Remember all elements are position:static by default
  • If none of it's parent have a position set, then it will use the <html> tag (window)

Fixed Positioning

  • Fixed positioning places the element using the window available
  • Use position:fixed; in CSS
  • Position will not change, even if you scroll!!!
