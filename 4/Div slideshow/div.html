Block & Inline Elements

Most elements in HTML fall into two different types
  • Inline
  • Block



    • First lets look at inline elements!

How to Elements behave on an HTML page?

  • These elements fall onto the page just like words in a book
  • They start at the top left and move along until the top is full
  • After this they move down a line and start again
These are called "inline" elements, because they move in a line

Examples


Inline Element Continued

  • Used for text content mostly
  • Will be as wide as the text it contains


    • some text

  • Cannot apply size ( width & height )
  • Cannot apply margins (top & bottom )
  • Can apply padding to left and right also

Block Types

  • These elements fall onto the page just like words in a book again (from the top down)
  • They take up the as much width as they can!!
  • They start at the top left and any other element that comes after them, must start on the next line

Some Examples

Block elements continued

  • Used for holding a specific piece of information
    • e.g. a paragraph of text or a heading
  • Can give it a width and a height

  • If no height is given, as high as the elements inside it

  • If no width is given, as wide as the block IT is inside

  • Can give it margins or padding on all sides

So how to divide up the page??

  • Since block elements can be changed by giving them a width and a height, they are very useful for making up the design of your page!

  • Block elements usually have a meaning given to them, such as the purpose of <p> is for paragraphs and <h1> is for headings

  • If we want a block to use just for organising our pages layout that has no other special meaning, we use the <DIV> tag

What is a div tag?

Like most other HTML elements, it has an opening and a closing tag

  • Just a block element!
  • Useful for dividing up your page into sections
  • A way to group together these sections
